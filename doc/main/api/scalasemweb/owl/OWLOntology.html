OWLOntology

class OWLOntology extends View with Set[OWLEntity] with SetLike[OWLEntity, OWLOntology]

An instance of an OWL ontology. The SSW OWL package is designed as a set of views on triple sets, the OWL ontology object views all triples in the ontology.
